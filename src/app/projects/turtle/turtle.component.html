Mira Hub

The Mira Hub (Nick Name Leatherback Turtle) is one of the main projects I developed while working in the mira team. A quick description of Mira Hub:

The hub will automatically process a video once it detects a usb device with new videos is plugged in. It can handle tasks like video encode/decode, video/audio muxing, video compression, etc. It then uploads the processed videos along with the original videos to a selected web drive (eg. Google drive, Baidu drive).

Here is a detailed user's manual.

Mira Hub is based on a Nvidia Jetson Nano. Software is mostly implemented in python and shell scripts, the video tasks are implemented with GStreamer. I tried to use FFmpeg, but it looks like the Nvidia GPU hardware acceleration is not supported for FFmpeg.

The display and UI is implemented with a system service written in python, based on a bettrey panel wtih a INA219 screen on it.

I also partnered up with another coop student from SJTU(Shanghai Jiao Tong University) to design a case for the hub. We decided to 3D print a case, since there is a pretty big 3D-printing lab in SJTU.
